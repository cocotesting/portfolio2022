Encyclopedia

Introduction

A Wikipedia-like online encyclopedia.

  1. Each encyclopedia entry can be viewed by visiting that entry’s page.
  2. Able to search for encyclopedia entrie by typing the query into the search box.
  3. Able to create new or edit existing encyclopedia entries within the web app.
  4. Show random encyclopedia entries by clicking "Random Page".
